\subsection{HSV (Hue, Saturation, Value) Color Model}
In the field of computer vision, selecting an appropriate color space is critical for the accuracy of object detection under varying environmental conditions. The HSV (Hue, Saturation, Value) color model is a cylindrical representation of colors that more closely aligns with human visual perception compared to the standard RGB model. While RGB defines colors as a combination of primary light sources, it is highly sensitive to changes in brightness, shadows, and external lighting.The HSV model decouples color information from intensity, using three distinct components:
\begin{itemize}
    \item Hue ($H$): Represents the dominant wavelength or the actual "color" (e.g., Red, Yellow, Green) expressed in degrees from $0^{\circ}$ to $360^{\circ}$.
    \item Saturation ($S$): Indicates the vibrancy or purity of the color, ranging from $0$ to $100\%$.
    \item Value ($V$): Represents the brightness or luminance of the color.
\end{itemize}
